6. NULL字段的小知识点整理

应用场景：
大数据中进来两张表table1和table2
table1中有数据：
0 1 1 …null null 1KW
table1中有数据：
0 1 1 …null null 1000KW
在做 table1 left join table2 on table1.xxx=table2.xxx时null数据匹配的时候null要匹配1KW*1000KW磁盘爆掉
所以在做on字段的时候要小心，先做下数据过滤(xxx匹配字段 not null)
